Transaction Setup


The transaction setup page will vary based upon the menu that is configured and the transaction currently selected.

The upper portion of the screen indicates the transactions that have been configure in the menu. Selecting a row here will display (in the lower portion) the configurable parameters of that transaction (i.e. on the Receiving transaction there is an option "Order Days After View" which affects the list of receipts).

Note - changes made here are only picked up by the devices once they have logged out of the ADC client
