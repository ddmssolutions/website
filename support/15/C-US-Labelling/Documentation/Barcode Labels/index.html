Barcode Labels

The Barcode Label table is used to setup the various type of barcode label that you will print, you associate the Business Central data with the printers label format as well as configure how and where the data in Business Central is sent to the printers


Code

Unique identifier for use within Business Central


Description

Informational purposes only


Type

This reqpresents where the usage of the label and is used when setting up the other parts of the system, i.e. Container Type setup will only show you the Container labels.


Print Method

Determines the interface used and the output format of the label:

  • Report - this will run the Report ID specified and output as per any other Business Central report
  • Loftware PAS - A text (.pas) file will be generated following the PAS format requirement of lofware. Loftware should be configured to monitor a directoty for these files and process / send them to the printers
  • Bartender DAT - A text (.dat) file will be generated following the Bartender requiremtns. Bartender should be configured to monitor for and process these files

    • Label Format File


    Report ID

    The Object ID of the Business Central report that will be used if Print Method is Report
