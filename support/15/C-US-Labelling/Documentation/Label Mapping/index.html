Label Mapping

Label mapping allows you to configure the data that is sent to the printed (via API, PAS, or DAT files). At minimum the Label Data table (14142141) must be sent with at least one fiel, however mulitple fields from that table or related tables can also be sent


Node Name

For Field type Data Sources this Node Name is wrote before the value of the Data Source


Data Source Type

Informational


Data Source

The Field or Table name from which the data is coming


Link to Table

Wehn Data Source Type is table this shows the parent table to which this node is linked


Table Relation

Used to specify how a sub table is linked to its parent in order to retrieve a single record (and thus data) for exporting


Data Type

Informational purposes relating to the field being exported


Format String

If populated the data retried from Business Central will be formatted using this string before being sent. This string uses the standard Business Central Format Property
