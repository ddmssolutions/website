Barcode Printers

Each physical printer that barcode labels are required to be printed to should be listed here


ID

Unique identifier


Name

Optionally used with Loftware / Bartender integration (and sent as part of the label data)


Share Name

Used for Bartender integration


Model

Descriptive purposes only


Info.

Descriptive purposes only for use within Business Central


API URL

The URL of the REST API that will be used to connect and send data to the printer


API Username

Used with API URL


API Password

Used with API URL


Drop File Folder Name

When Loftware PAS, Bartender DAT, or Report (Business Central reports (pdfs)) as the Print Method the files will created in this folder
