Message Log


The message log contains all of the errors that are shown to each device. Both the error the user saw (i.e. "Invalid Location") and the more details explanation of that error. This message log should have its older data periodically removed, this can be achieved either manually via the Delete action or by the use of the Job Queue Entry routine that runs codeunit 14142104 "Clear Message Log" (this job queue entry uses the Paramenter String as a dateformula to determine which messages are removed and defaults to 1 week if not specified).

User ID

The Session ID of the device that received the error

Message Date & Time

When the error was received

Transaction No.

The table no. of the transaction

Transaction Name

The name of the transaction

Full Error

The full error information that is logged (this is the additional information that is displayed on the device when the Error Key is pressed
