Setup


System Indicator

When populate this test will appear on the upper part of ADC Modern Client. It can be used to help indicate to users if they are in the test, live, development... systems. Simple HTML markup language can be used here as well as plain text

List Key

The List Key is used to access the various lists of possible data values within conversation (i.e. List of Locations, List of Pick Notes...)

Error Key

Errors shown to users on the devices are designed to be short and generic (i.e. Invalid Location), in the background however more detailed information is logged. To access that extra information the user should press this Error Key to access that logged error on the device without having to access Business Central via the modern client

UCC Variable Length Delimiter

UCC / GS128 Barcodes that contain multiple variable length fields (i.e. Lot No. and Serial No.) use a FNC1 character to denote the end of the field. When in use the devices barcode scanners must be configured to replace the FNC1 character with an ASCII char so that it is readable & parsable by Business Central.
