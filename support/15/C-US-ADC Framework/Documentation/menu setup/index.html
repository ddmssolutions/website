Menu Setup

The menu that appears on the ADC Modern Client is configured here and applies to all users


Description

This is the text that will appear within the ADC Modern Client menu


Transaction Table ID

If left blank this indicates that record is a sub menu and should contain one or more records indented underneath it. If populated this represents the Table No. within the application that has been built to the ADC Standards


Icon Filename

The name of a graphics file can be entered here of a filename that exists within the ADC websites Images folder. This image will then be rendered against the menu item within the ADC Modern Client. Custom icons can be added to the images folder and referenced in the same manner.

The following icon files are included within the images folder as standard and can be used in the above field:
